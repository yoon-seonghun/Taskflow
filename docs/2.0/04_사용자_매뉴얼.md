## 1. 시스템 소개

### 1.1 TaskFlow란?

TaskFlow는 Notion 스타일의 업무 관리 시스템으로, 팀 구성원 간 업무 현황을 실시간으로 공유하고 협업할 수 있는 웹 애플리케이션입니다.

**핵심 특징**
- **Compact UI**: 한 화면에 최대한 많은 정보를 표시하는 밀도 높은 인터페이스
- **다중 뷰**: 테이블, 칸반, 리스트 3가지 뷰 타입 지원
- **실시간 협업**: 여러 사용자가 동시에 작업 가능
- **유연한 속성 시스템**: 업무에 필요한 속성을 자유롭게 추가/수정
- **모바일 최적화**: PC와 모바일에서 모두 사용 가능한 반응형 디자인

### 1.2 주요 기능 개요

| 기능 | 설명 |
|------|------|
| 보드 관리 | 프로젝트별로 보드를 생성하고 관리 |
| 업무 관리 | 업무 등록, 수정, 완료, 삭제 |
| 뷰 타입 전환 | 테이블/칸반/리스트 뷰 간 자유로운 전환 |
| 공유 | 보드를 팀원과 공유하고 협업 |
| 댓글 | 업무별 의견 첨부 및 커뮤니케이션 |
| 파일 첨부 | 업무에 관련 파일 첨부 (이미지, 문서 등) |
| 필터링 | 상태별 업무 필터링 (전체/대기/진행중/지연/보류) |
| 이력 관리 | 완료된 업무 및 작업 템플릿 이력 조회 |

### 1.3 지원 브라우저

- Chrome 최신 버전 (권장)
- Firefox 최신 버전
- Safari 최신 버전
- Edge 최신 버전

**권장 환경**
- 해상도: 1366x768 이상
- 모바일: iOS 12 이상, Android 8 이상

## 2. 시작하기

### 2.1 로그인 방법

1. 브라우저에서 TaskFlow URL에 접속합니다.
2. 로그인 페이지에서 아이디와 비밀번호를 입력합니다.
3. "아이디 저장" 체크박스를 선택하면 다음 로그인 시 아이디가 자동으로 입력됩니다.
4. "로그인" 버튼을 클릭합니다.

**입력 규칙**
- 아이디: 4자 이상
- 비밀번호: 8자 이상

**세션 만료**
- 로그인 상태는 7일간 유지됩니다.
- 세션이 만료되면 자동으로 로그인 페이지로 이동합니다.

### 2.2 화면 구성 설명

TaskFlow는 크게 헤더, 사이드바, 메인 영역으로 구성됩니다.

```
┌─────────────────────────────────────────────────────────┐
│                      Header                              │
├──────────┬──────────────────────────────────────────────┤
│          │                                               │
│ Sidebar  │              Main Panel                       │
│  (Nav)   │        (테이블/칸반/리스트 뷰)                  │
│          │                                               │
│          │                          ┌──────────────────┐ │
│          │                          │   Slide-over    │ │
│          │                          │   Panel         │ │
│          │                          │   (상세/편집)    │ │
│          │                          └──────────────────┘ │
└──────────┴──────────────────────────────────────────────┘
```

#### 2.2.1 헤더 (Header)

- 왼쪽: 햄버거 메뉴 (모바일에서 사이드바 토글)
- 가운데: TaskFlow 로고
- 오른쪽: 사용자 정보 및 로그아웃 버튼

#### 2.2.2 사이드바 (Sidebar)

**메뉴 구조**
- 📋 업무 페이지 (메인 화면)
- ⚠️ 지연 업무 (마감일이 지난 업무)
- ⏸️ 보류 업무 (보류 상태 업무)
- ✅ 완료 작업 (완료된 업무 목록)
- 🗑️ 삭제된 작업
- ─────────────────
- 📝 작업 등록 (작업 템플릿 관리)
- 📊 이력관리
- ─────────────────
- 👤 사용자 등록
- 👥 공유 사용자
- ─────────────────
- 🏢 부서 관리
- 📁 그룹 관리
- 📋 보드 관리

**보드 목록**
사이드바 하단에 보드 목록이 3개 섹션으로 표시됩니다:
- **내 보드**: 내가 만든 보드 (공유하지 않은 것)
- **공유해준 보드**: 내가 만들고 다른 사람과 공유한 보드
- **공유받은 보드**: 다른 사람이 나에게 공유한 보드

각 섹션은 클릭하여 접기/펼치기가 가능합니다.

#### 2.2.3 메인 영역

현재 선택된 메뉴의 내용이 표시됩니다.

#### 2.2.4 슬라이드오버 패널

업무를 클릭하면 화면 오른쪽에서 슬라이드하여 나타나는 상세 편집 패널입니다.
- **PC**: 화면 오른쪽에 패널이 슬라이드되어 표시
- **모바일**: 전체 화면으로 상세 페이지가 표시

### 2.3 로그아웃

1. 헤더 오른쪽의 사용자 아이콘을 클릭합니다.
2. "로그아웃" 버튼을 클릭합니다.
3. 로그인 페이지로 자동 이동합니다.

## 3. 보드 관리

### 3.1 보드란?

보드는 업무를 그룹화하는 최상위 단위입니다. 프로젝트, 팀, 업무 유형별로 보드를 만들어 관리할 수 있습니다.

**예시**
- "웹사이트 리뉴얼 프로젝트"
- "개발팀 일일 업무"
- "고객 지원 요청"

### 3.2 보드 생성

1. 사이드바에서 "보드 관리" 메뉴를 클릭합니다.
2. 우측 상단의 "새 보드" 버튼을 클릭합니다.
3. 보드 정보를 입력합니다:
   - **보드 이름** (필수): 보드의 제목
   - **설명** (선택): 보드의 용도나 설명
   - **색상**: 보드를 구분하기 위한 색상 (10가지 색상 중 선택)
4. "생성" 버튼을 클릭합니다.

### 3.3 보드 수정

1. "보드 관리" 페이지에서 수정할 보드의 연필 아이콘(✏️)을 클릭합니다.
2. 보드 정보를 수정합니다.
3. "저장" 버튼을 클릭합니다.

### 3.4 보드 삭제

1. "보드 관리" 페이지에서 삭제할 보드의 휴지통 아이콘(🗑️)을 클릭합니다.
2. 삭제 확인 다이얼로그가 표시됩니다.

**미완료 업무가 있는 경우**
- 옵션 1: **업무 이관 대상자 선택** - 선택한 사용자의 새 보드로 미완료 업무가 이동됩니다.
- 옵션 2: **강제 삭제** - 미완료 업무를 포함하여 모두 삭제됩니다. (복구 불가능)

3. "삭제" 버튼을 클릭하여 확정합니다.

### 3.5 보드 색상 설정

보드 색상은 사이드바의 보드 목록에서 작은 원으로 표시됩니다. 보드를 빠르게 구분하는데 유용합니다.

**사용 가능한 색상**
- 파란색 (#3B82F6) - 기본값
- 초록색 (#10B981)
- 노란색 (#F59E0B)
- 빨간색 (#EF4444)
- 보라색 (#8B5CF6)
- 분홍색 (#EC4899)
- 인디고 (#6366F1)
- 청록색 (#14B8A6)
- 주황색 (#F97316)
- 회색 (#6B7280)

### 3.6 뷰 타입 변경

보드의 업무 목록을 3가지 형태로 볼 수 있습니다.

1. "업무 페이지"에서 보드를 선택합니다.
2. 보드 선택 드롭다운 옆에 있는 뷰 타입 버튼을 클릭합니다:
   - **테이블** (☰): 스프레드시트 형태
   - **칸반** (▦): 컬럼별 카드 배치
   - **리스트** (≡): 단순 목록 형태

선택한 뷰 타입은 브라우저에 저장되어 다음에도 유지됩니다.

## 4. 업무 관리

### 4.1 업무 등록 (신규 업무 창)

#### 4.1.1 기본 등록 방법

1. "업무 페이지"에서 보드를 선택합니다.
2. 상단의 "새 업무를 입력하세요..." 입력 필드를 클릭합니다.
3. 업무 내용을 입력합니다.
4. "등록" 버튼을 클릭하거나 Enter 키를 누릅니다.

#### 4.1.2 작업 템플릿 자동완성

자주 사용하는 업무 내용은 자동완성 기능을 통해 빠르게 입력할 수 있습니다.

1. 입력 필드에 2글자 이상 입력합니다.
2. 300ms 후 자동으로 등록된 작업 템플릿을 검색합니다.
3. 드롭다운 목록에서 원하는 템플릿을 선택합니다.
4. 선택하면 입력 필드에 자동으로 채워집니다.

**자동완성 정보**
- 최대 10개 결과 표시
- 사용 횟수가 함께 표시됩니다 (예: "사용 5회")
- 템플릿을 선택하면 사용 횟수가 자동 증가합니다

### 4.2 업무 목록 조회

#### 4.2.1 필터 탭

업무 목록 상단에 5개의 필터 탭이 있습니다:

| 탭 | 설명 | 표시 업무 |
|----|------|----------|
| 전체 | 모든 활성 업무 | 완료/삭제를 제외한 모든 업무 |
| 대기 | 시작 전 | 상태가 "시작전"인 업무 |
| 진행중 | 진행 중 | 상태가 "진행중"인 업무 |
| 지연 ⚠️ | 지연된 업무 | 완료 예정일이 오늘보다 이전인 업무 |
| 보류 | 보류 상태 | 상태가 "보류"인 업무 |

각 탭에는 해당 업무 개수가 표시됩니다.

#### 4.2.2 업무 정보

테이블 뷰에서 각 업무는 다음 정보를 표시합니다:
- 업무 내용
- 시작 시간
- 완료 시간
- 담당자
- 카테고리
- 상태
- 우선순위
- 그룹
- 댓글 수 (💬 아이콘)

### 4.3 업무 상세 편집 (슬라이드오버 패널)

#### 4.3.1 패널 열기

- **PC**: 업무 행을 클릭하면 우측에서 패널이 슬라이드됩니다.
- **모바일**: 업무를 탭하면 전체 화면 편집 페이지로 이동합니다.

#### 4.3.2 패널 구조 (PC)

PC에서 패널은 3개 영역으로 나뉘며, 각 영역의 폭을 조절할 수 있습니다:

```
┌──────────┬────────────────────┬──────────┐
│          │                    │          │
│  속성    │   내용 + 파일      │  댓글    │
│          │                    │          │
│  (240px) │                    │  (280px) │
│          │                    │          │
└──────────┴────────────────────┴──────────┘
     ↕              ↕                ↕
  리사이즈       리사이즈         리사이즈
  핸들           핸들             핸들
```

**폭 조절 방법**
1. 영역 사이의 회색 구분선에 마우스를 올립니다.
2. 마우스 커서가 양방향 화살표로 변합니다.
3. 드래그하여 원하는 폭으로 조절합니다.
4. 조절한 폭은 브라우저에 저장되어 다음에도 유지됩니다.

**폭 제한**
- 속성 패널: 최소 180px ~ 최대 400px
- 댓글 패널: 최소 200px ~ 최대 450px

#### 4.3.3 속성 영역

업무의 모든 속성을 편집할 수 있습니다.

**기본 속성**
- **제목**: 업무의 제목
- **상태**: 시작전 / 진행중 / 보류 / 완료 / 삭제
- **우선순위**: 긴급 / 높음 / 보통 / 낮음
- **담당자**: 업무 담당자 선택
- **시작 시간**: DateTimePicker로 선택
- **완료 예정일**: DateTimePicker로 선택
- **카테고리**: 드롭다운 선택
- **그룹**: 드롭다운 선택

**인라인 편집**
모든 속성은 클릭하면 즉시 편집 가능하며, 변경사항은 자동으로 저장됩니다.

#### 4.3.4 내용 영역

**마크다운 에디터**
- 마크다운 형식으로 상세 내용을 작성할 수 있습니다.
- 실시간 미리보기 지원
- 이미지, 링크, 코드 블록 등 마크다운 문법 사용 가능

**자동 저장**
- 입력 후 자동으로 저장됩니다.
- 저장 중일 때 "저장 중..." 표시

**최소 높이**: 100% (패널 전체 높이)

#### 4.3.5 파일 첨부 영역

내용 영역 하단에 파일 첨부 영역이 있습니다. (자세한 내용은 [8. 파일 첨부](#8-파일-첨부) 참고)

#### 4.3.6 댓글 영역

오른쪽 댓글 패널에서 업무에 대한 의견을 주고받을 수 있습니다. (자세한 내용은 [7. 댓글 기능](#7-댓글-기능) 참고)

#### 4.3.7 패널 헤더

패널 상단 헤더에는 다음 버튼들이 있습니다:

| 버튼 | 기능 | 표시 조건 |
|------|------|----------|
| 이관 | 업무를 다른 사용자의 보드로 이관 | 이관 권한이 있을 때 |
| 공유 | 업무를 다른 사용자와 공유 | 공유 권한이 있을 때 |
| 완료 | 업무를 완료 처리 | 상태가 완료/삭제가 아닐 때 |
| 삭제 | 업무를 삭제 처리 | 상태가 삭제가 아닐 때 |
| X | 패널 닫기 | 항상 |

**수정자 정보**
헤더 하단에 최종 수정자와 수정 시간이 표시됩니다.
- 예: "홍길동님이 12월 24일 14:30에 수정"

#### 4.3.8 모바일 레이아웃

모바일에서는 3개 탭으로 구성됩니다:

| 탭 | 내용 |
|----|------|
| 속성 | 업무 속성 편집 |
| 내용 | 마크다운 에디터 + 파일 첨부 |
| 댓글 | 댓글 목록 및 입력 |

탭을 전환하며 각 영역에 접근할 수 있습니다.

### 4.4 인라인 편집 방법

테이블 뷰에서 패널을 열지 않고 바로 편집할 수 있습니다.

1. 테이블에서 편집하려는 셀을 클릭합니다.
2. 편집 UI가 나타납니다:
   - 텍스트: 인라인 입력 필드
   - 날짜: DateTimePicker
   - 선택형: 드롭다운
3. 값을 변경합니다.
4. 다른 곳을 클릭하거나 Enter를 누르면 자동 저장됩니다.

### 4.5 상태 변경

업무 상태는 5가지입니다:

| 상태 | 설명 | 색상 |
|------|------|------|
| 시작전 | 아직 시작하지 않은 업무 | 회색 |
| 진행중 | 현재 진행 중인 업무 | 파란색 |
| 보류 | 일시적으로 중단된 업무 | 주황색 |
| 완료 | 완료된 업무 | 초록색 |
| 삭제 | 삭제된 업무 | 빨간색 |

**변경 방법**
1. 테이블 뷰에서 상태 셀을 클릭하거나
2. 상세 패널에서 상태 드롭다운을 클릭하거나
3. 칸반 뷰에서 카드를 다른 컬럼으로 드래그 앤 드롭

### 4.6 우선순위 설정

업무 우선순위는 4가지입니다:

| 우선순위 | 설명 | 색상 |
|----------|------|------|
| 긴급 | 즉시 처리 필요 | 빨간색 |
| 높음 | 중요한 업무 | 주황색 |
| 보통 | 일반 업무 | 파란색 |
| 낮음 | 여유있게 처리 가능 | 회색 |

**변경 방법**
- 상세 패널 또는 테이블 뷰에서 우선순위 드롭다운 선택
- 칸반 뷰에서 "우선순위" 그룹 기준으로 변경 시 카드 드래그 앤 드롭

### 4.7 담당자 지정

1. 담당자 필드를 클릭합니다.
2. 사용자 검색/선택 드롭다운이 나타납니다.
3. 담당자를 선택하거나 검색합니다.
4. 선택하면 자동으로 저장됩니다.

**미지정 상태**
담당자를 지정하지 않으면 "미지정"으로 표시됩니다.

### 4.8 시작/완료 시간 설정

1. 시작 시간 또는 완료 예정일 필드를 클릭합니다.
2. DateTimePicker가 나타납니다.
3. 날짜와 시간을 선택합니다.
4. 확인하면 자동으로 저장됩니다.

**지연 업무 자동 표시**
완료 예정일이 오늘보다 이전이고 상태가 "완료"가 아닌 업무는:
- "지연" 필터 탭에 표시됩니다.
- 사이드바의 "지연 업무" 메뉴에도 표시됩니다.

### 4.9 업무 완료 처리

#### 방법 1: 완료 버튼 (상세 패널)
1. 업무를 클릭하여 상세 패널을 엽니다.
2. 상단의 "완료" 버튼을 클릭합니다.
3. 확인 다이얼로그에서 "완료"를 클릭합니다.

#### 방법 2: 상태 변경
상태를 "완료"로 변경하면 자동으로 완료 처리됩니다.

**완료 처리 효과**
- 상태가 "완료"로 변경됩니다.
- 현재 목록에서 Hidden 처리됩니다.
- 페이지 하단의 "완료된 업무" 영역에 축소 표시됩니다.
- "완료 작업" 메뉴에서 조회 가능합니다.

### 4.10 업무 삭제 처리

1. 업무를 클릭하여 상세 패널을 엽니다.
2. 상단의 "삭제" 버튼을 클릭합니다.
3. 확인 다이얼로그에서 "삭제"를 클릭합니다.

**삭제 처리 효과**
- 상태가 "삭제"로 변경됩니다.
- 현재 목록에서 Hidden 처리됩니다.
- "삭제된 작업" 메뉴에서 조회 가능합니다.
- 영구 삭제는 아니므로 복원 가능합니다.

### 4.11 완료/삭제 업무 Hidden 처리

"전체" 필터 탭에서 완료 또는 삭제된 업무는 목록에서 자동으로 숨겨집니다.

**Hidden 규칙 (당일 처리 기준)**
- 오늘 날짜에 완료/삭제된 항목만 하단에 표시됩니다.
- 이전 날짜에 완료/삭제된 항목은 표시되지 않습니다.

**확인 방법**
1. 업무 목록 페이지 맨 하단을 확인합니다.
2. "완료된 업무 (N건)" 형태로 축소되어 표시됩니다.
3. 클릭하면 확장되어 완료/삭제 목록이 표시됩니다.
4. 다시 클릭하면 축소됩니다.

**표시 정보**
- 작업 내용
- 작업 결과 (완료/삭제)
- 처리 시간

## 5. 뷰 사용법

### 5.1 테이블 뷰

스프레드시트와 유사한 형태로, 한 눈에 많은 정보를 볼 수 있습니다.

#### 5.1.1 특징
- 컬럼별로 속성 표시
- 인라인 편집 지원
- 컬럼 정렬 가능
- 고밀도 UI (한 화면에 많은 업무 표시)

#### 5.1.2 컬럼 정렬

1. 컬럼 헤더를 클릭합니다.
2. 오름차순/내림차순으로 정렬됩니다.
3. 다시 클릭하면 정렬 방향이 바뀝니다.

#### 5.1.3 필터링

상단의 필터 탭을 사용하여 업무를 필터링할 수 있습니다.

#### 5.1.4 적합한 용도
- 다중 속성을 한 번에 비교할 때
- 대량의 업무를 빠르게 스캔할 때
- 데이터 입력이 많을 때

### 5.2 칸반 뷰

컬럼별로 카드를 배치하여 워크플로우를 시각화합니다.

#### 5.2.1 특징
- 상태/우선순위/담당자/그룹별 그룹핑
- 드래그 앤 드롭으로 속성 변경
- 카드 기반 UI
- 진행 상황을 시각적으로 파악

#### 5.2.2 그룹 기준 변경

1. 칸반 뷰 상단의 "그룹 기준" 드롭다운을 클릭합니다.
2. 원하는 기준을 선택합니다:
   - **상태**: 시작전 / 진행중 / 보류 / 완료 / 삭제
   - **우선순위**: 긴급 / 높음 / 보통 / 낮음
   - **담당자**: 사용자별 그룹핑 + 미지정
   - **그룹**: 그룹별 그룹핑 + 미분류

#### 5.2.3 드래그 앤 드롭으로 변경

1. 카드를 클릭하고 드래그합니다.
2. 원하는 컬럼으로 드래그합니다.
3. 드롭하면 해당 속성이 자동으로 변경됩니다.

**예시**
- 상태 기준 칸반에서 "진행중" 컬럼으로 드래그하면 → 상태가 "진행중"으로 변경
- 담당자 기준 칸반에서 "홍길동" 컬럼으로 드래그하면 → 담당자가 "홍길동"으로 변경

#### 5.2.4 컬럼 정보

각 컬럼은 다음 정보를 표시합니다:
- 컬럼 제목 (상태/우선순위/담당자/그룹 이름)
- 해당 컬럼의 업무 개수
- 색상 구분

#### 5.2.5 적합한 용도
- 상태 기반 워크플로우 관리
- 진행 상황을 한눈에 파악
- 업무 이동이 빈번한 경우

### 5.3 리스트 뷰

단순한 목록 형태로, 모바일에 최적화되어 있습니다.

#### 5.3.1 특징
- 간결한 인터페이스
- 빠른 스캔
- 모바일 친화적
- 핵심 정보만 표시

#### 5.3.2 표시 정보
- 업무 제목
- 상태 배지
- 우선순위 배지
- 담당자
- 완료 예정일

#### 5.3.3 적합한 용도
- 모바일에서 업무 확인
- 간단한 업무 목록 확인
- 빠른 업무 추가/완료 처리

## 6. 공유 기능

### 6.1 보드 공유

보드를 팀원과 공유하여 협업할 수 있습니다.

#### 6.1.1 공유 사용자 추가

1. "보드 관리" 메뉴로 이동합니다.
2. 공유할 보드의 공유 아이콘(👥)을 클릭합니다.
3. "공유 관리" 모달이 열립니다.
4. 상단의 사용자 검색 필드에 이름 또는 부서를 입력합니다.
5. 검색 결과에서 사용자를 선택합니다.
6. 권한을 선택합니다:
   - **조회(VIEW)**: 업무를 볼 수만 있음
   - **편집(EDIT)**: 업무 조회 및 수정 가능
   - **전체(FULL)**: 업무 조회, 수정, 삭제 가능
7. "추가" 버튼을 클릭합니다.

#### 6.1.2 공유 사용자 제거

1. "공유 관리" 모달에서 제거할 사용자를 찾습니다.
2. 사용자 행의 "제거" 버튼을 클릭합니다.
3. 확인 다이얼로그에서 "확인"을 클릭합니다.

#### 6.1.3 권한 설정

| 권한 | 조회 | 편집 | 삭제 | 공유 관리 |
|------|------|------|------|-----------|
| 조회(VIEW) | ✅ | ❌ | ❌ | ❌ |
| 편집(EDIT) | ✅ | ✅ | ❌ | ❌ |
| 전체(FULL) | ✅ | ✅ | ✅ | ❌ |
| 소유자(OWNER) | ✅ | ✅ | ✅ | ✅ |

**참고**: 공유 사용자 관리는 보드 소유자만 가능합니다.

### 6.2 업무 공유

개별 업무를 다른 사용자와 공유할 수 있습니다.

#### 6.2.1 업무 공유 방법

1. 업무를 클릭하여 상세 패널을 엽니다.
2. 상단의 "공유" 버튼을 클릭합니다.
3. "업무 공유" 모달이 열립니다.
4. 사용자를 검색하고 선택합니다.
5. 권한을 선택합니다 (조회/편집/전체).
6. "추가" 버튼을 클릭합니다.

#### 6.2.2 공유 해제

1. "업무 공유" 모달에서 제거할 사용자를 찾습니다.
2. "제거" 버튼을 클릭합니다.

### 6.3 업무 이관

업무를 다른 사용자의 보드로 이관할 수 있습니다.

#### 6.3.1 이관 방법

1. 업무를 클릭하여 상세 패널을 엽니다.
2. 상단의 "이관" 버튼을 클릭합니다.
3. "업무 이관" 모달이 열립니다.
4. 이관 대상 사용자를 선택합니다.
5. 대상 보드를 선택합니다 (사용자의 보드 목록이 표시됨).
6. "이관" 버튼을 클릭합니다.

#### 6.3.2 이관 효과

- 업무가 현재 보드에서 제거됩니다.
- 대상 사용자의 선택한 보드에 추가됩니다.
- 업무의 소유자가 대상 사용자로 변경됩니다.
- 이관 후 원래 보드에서는 조회할 수 없습니다.

## 7. 댓글 기능

업무에 대한 의견을 주고받을 수 있습니다.

### 7.1 댓글 작성

1. 업무를 클릭하여 상세 패널을 엽니다.
2. PC: 오른쪽 댓글 패널 / 모바일: "댓글" 탭을 선택합니다.
3. 하단의 댓글 입력 필드에 내용을 작성합니다.
4. "등록" 버튼을 클릭하거나 Ctrl+Enter를 누릅니다.

### 7.2 댓글 수정

1. 댓글 위에 마우스를 올립니다.
2. 연필 아이콘(✏️)을 클릭합니다.
3. 내용을 수정합니다.
4. "저장" 버튼을 클릭하거나 Enter를 누릅니다.

**수정 표시**
수정된 댓글에는 "(수정됨)" 표시가 추가됩니다.

### 7.3 댓글 삭제

1. 댓글 위에 마우스를 올립니다.
2. 휴지통 아이콘(🗑️)을 클릭합니다.
3. 확인 다이얼로그에서 "삭제"를 클릭합니다.

### 7.4 댓글 목록 확인

#### 7.4.1 댓글 수 표시

- 테이블 뷰의 "리플" 컬럼에 댓글 수가 표시됩니다.
- 상세 패널의 댓글 탭에 "(N)" 형태로 표시됩니다.

#### 7.4.2 댓글 정보

각 댓글은 다음 정보를 표시합니다:
- 작성자 아바타 (이름의 첫 글자)
- 작성자 이름
- 작성 시간 (상대 시간: "방금 전", "5분 전", "2시간 전" 등)
- 댓글 내용

### 7.5 시간 표시 형식

| 경과 시간 | 표시 형식 |
|-----------|----------|
| 1분 미만 | 방금 전 |
| 1시간 미만 | N분 전 |
| 24시간 미만 | N시간 전 |
| 7일 미만 | N일 전 |
| 7일 이상 | 12월 24일 14:30 |

## 8. 파일 첨부

업무에 관련 파일을 첨부할 수 있습니다.

### 8.1 파일 업로드 방법

#### 방법 1: 파일 선택

1. 업무 상세 패널의 파일 첨부 영역을 찾습니다.
2. "파일 추가" 버튼을 클릭합니다.
3. 파일 선택 다이얼로그에서 파일을 선택합니다.
4. 여러 개 선택 가능합니다 (Ctrl/Cmd + 클릭).

#### 방법 2: 드래그 앤 드롭

1. 파일 탐색기에서 파일을 선택합니다.
2. 파일 첨부 영역으로 드래그합니다.
3. 영역이 파란색으로 강조됩니다.
4. 파일을 드롭합니다.

### 8.2 지원 파일 형식

#### 8.2.1 이미지

- JPEG, JPG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- SVG (.svg)
- WebP (.webp)
- BMP (.bmp)

#### 8.2.2 문서

- PDF (.pdf)
- Microsoft Word (.doc, .docx)
- Microsoft Excel (.xls, .xlsx)
- Microsoft PowerPoint (.ppt, .pptx)
- 텍스트 파일 (.txt)

#### 8.2.3 압축 파일

- ZIP (.zip)
- RAR (.rar)
- 7Z (.7z)

#### 8.2.4 기타

- JSON (.json)
- XML (.xml)
- CSV (.csv)

### 8.3 파일 크기 제한

- 최대 파일 크기: **10MB**
- 최대 첨부 개수: **10개**

크기를 초과하거나 개수 제한을 넘으면 오류 메시지가 표시됩니다.

### 8.4 파일 다운로드

1. 파일 목록에서 다운로드할 파일을 찾습니다.
2. 파일 이름을 클릭하거나 다운로드 아이콘을 클릭합니다.
3. 브라우저의 다운로드 폴더에 저장됩니다.

### 8.5 파일 삭제

1. 파일 목록에서 삭제할 파일을 찾습니다.
2. 휴지통 아이콘(🗑️)을 클릭합니다.
3. 확인 다이얼로그에서 "삭제"를 클릭합니다.

**주의**: 삭제된 파일은 복구할 수 없습니다.

### 8.6 파일 목록 보기

#### 8.6.1 접기/펼치기

파일 첨부 영역은 기본적으로 펼쳐져 있으며, 헤더를 클릭하여 접을 수 있습니다.

- 펼쳐짐: 파일 목록이 표시됨
- 접힘: 헤더만 표시됨 (파일 개수는 계속 표시)

#### 8.6.2 스크롤 영역

파일이 많을 경우 최대 150px 높이로 제한되며, 스크롤하여 볼 수 있습니다.

## 9. 완료/이력 관리

### 9.1 완료 작업 조회

완료된 업무 목록을 조회할 수 있습니다.

#### 9.1.1 완료 작업 메뉴

1. 사이드바에서 "완료 작업" 메뉴를 클릭합니다.
2. 완료된 업무 목록이 표시됩니다.

#### 9.1.2 표시 정보

| 컬럼 | 설명 |
|------|------|
| 등록시간 | 업무 최초 등록 시간 |
| 완료시간 | 업무 완료 처리 시간 |
| 작업 내용 | 업무 제목/내용 |
| 작업자 | 완료 처리한 담당자 |

#### 9.1.3 필터/정렬

**기간 필터**
- 오늘
- 이번 주
- 이번 달
- 사용자 지정 (시작일 ~ 종료일)

**정렬**
- 기본: 완료시간 내림차순 (최근 완료된 것부터)

### 9.2 삭제된 작업 조회

1. 사이드바에서 "삭제된 작업" 메뉴를 클릭합니다.
2. 삭제된 업무 목록이 표시됩니다.
3. 완료 작업과 동일한 형식으로 표시됩니다.

### 9.3 이력관리 메뉴

작업 처리 이력과 작업 등록 이력을 조회할 수 있습니다.

#### 9.3.1 스위치 전환

페이지 상단에 스위치가 있습니다:
- **작업 처리 이력**: 완료/삭제된 업무의 처리 내역
- **작업 등록 이력**: 작업 템플릿의 등록/수정 내역

클릭하여 전환할 수 있습니다.

| 수정시간 | DATETIME | 마지막 수정 시간 |
| 삭제시간 | DATETIME | 삭제 처리 시간 (삭제된 경우) |

#### 9.3.3 작업 등록 이력 컬럼

| 컬럼 | 타입 | 설명 |

## 10. 자주 묻는 질문 (FAQ)

### 10.1 로그인 및 계정

**Q: 비밀번호를 잊어버렸어요.**

A: 관리자에게 문의하여 비밀번호를 재설정 받으세요. 현재 버전에서는 셀프 비밀번호 재설정 기능이 없습니다.

**Q: 아이디 저장 기능이 안전한가요?**

A: 아이디만 브라우저에 저장되며, 비밀번호는 저장되지 않습니다. 하지만 공용 PC에서는 사용을 권장하지 않습니다.

**Q: 로그인 상태가 얼마나 유지되나요?**

A: 7일간 유지됩니다. 7일 후에는 자동으로 로그아웃되어 재로그인이 필요합니다.

### 10.2 보드 관리

**Q: 보드를 몇 개까지 만들 수 있나요?**

A: 보드 개수에 제한은 없습니다. 단, 너무 많은 보드를 만들면 관리가 어려울 수 있으므로 적절한 개수로 유지하는 것을 권장합니다.

**Q: 보드를 삭제하면 업무도 모두 삭제되나요?**

A: 보드 삭제 시 옵션을 선택할 수 있습니다:
1. 업무 이관: 미완료 업무를 다른 사용자에게 이관
2. 강제 삭제: 모든 업무를 함께 삭제 (복구 불가능)

**Q: 보드 색상을 나중에 변경할 수 있나요?**

A: 네, 언제든지 "보드 관리" 메뉴에서 보드를 수정하여 색상을 변경할 수 있습니다.

### 10.3 업무 관리

**Q: 업무를 잘못 삭제했어요. 복원할 수 있나요?**

A: 네, "삭제된 작업" 메뉴에서 해당 업무를 찾아 상태를 변경하면 복원됩니다. 단, 강제 삭제된 경우에는 복원이 불가능합니다.

**Q: 업무의 속성을 변경하면 즉시 저장되나요?**

A: 네, TaskFlow는 자동 저장 방식을 사용합니다. 속성을 변경하면 즉시 서버에 저장됩니다.

**Q: 한 업무에 담당자를 여러 명 지정할 수 있나요?**

A: 현재 버전에서는 담당자를 1명만 지정할 수 있습니다. 여러 명이 작업하는 경우 업무를 공유하거나 댓글로 협업하세요.

**Q: 업무를 복사할 수 있나요?**

A: 현재 버전에서는 업무 복사 기능이 없습니다. 작업 템플릿 기능을 활용하여 비슷한 업무를 빠르게 등록할 수 있습니다.

### 10.4 뷰 사용

**Q: 칸반 뷰에서 카드 순서를 변경할 수 있나요?**

A: 현재 버전에서는 컬럼 간 이동(속성 변경)만 가능하며, 같은 컬럼 내에서 순서 변경은 지원하지 않습니다.

**Q: 테이블 뷰에서 컬럼을 숨길 수 있나요?**

A: 현재 버전에서는 컬럼 숨김 기능이 없습니다. 모든 속성이 항상 표시됩니다.

**Q: 선택한 뷰 타입이 저장되나요?**

A: 네, 브라우저에 저장되어 다음에 접속해도 마지막에 선택한 뷰 타입이 유지됩니다.

### 10.5 공유 및 권한

**Q: 공유받은 보드의 권한을 확인할 수 있나요?**

A: 네, 사이드바의 "공유받은 보드" 섹션에서 각 보드 옆에 권한이 표시됩니다 (조회/편집/전체).

**Q: 보드를 공유하면 모든 업무가 공유되나요?**

A: 네, 보드를 공유하면 해당 보드의 모든 업무를 공유받은 사용자가 볼 수 있습니다.

**Q: 업무를 개별적으로 공유할 수 있나요?**

A: 네, 업무 상세 패널의 "공유" 버튼을 사용하여 특정 업무만 공유할 수 있습니다.

### 10.6 댓글 및 파일

**Q: 댓글에 파일을 첨부할 수 있나요?**

A: 현재 버전에서는 댓글에 파일 첨부가 불가능합니다. 파일은 업무에만 첨부할 수 있습니다.

**Q: 10MB보다 큰 파일을 업로드하려면 어떻게 하나요?**

A: 현재 버전에서는 10MB가 최대 크기입니다. 큰 파일은 압축하거나 외부 클라우드 서비스를 사용하고 링크를 업무 내용에 기재하세요.

**Q: 업로드한 파일의 미리보기를 볼 수 없나요?**

A: 현재 버전에서는 파일 미리보기 기능이 없습니다. 다운로드하여 확인해야 합니다.

### 10.7 성능 및 기술

**Q: 인터넷 연결이 끊기면 어떻게 되나요?**

A: 인터넷이 끊기면 작업한 내용이 저장되지 않습니다. 연결이 복구되면 다시 시도할 수 있습니다.

**Q: 여러 사람이 동시에 같은 업무를 수정하면 어떻게 되나요?**

A: TaskFlow는 실시간 동기화를 지원합니다. 다른 사용자가 수정한 내용은 자동으로 반영됩니다. 충돌이 발생하면 충돌 해결 다이얼로그가 표시됩니다.

**Q: 모바일에서도 모든 기능을 사용할 수 있나요?**

A: 네, 모바일에서도 대부분의 기능을 사용할 수 있습니다. 단, PC에서 더 효율적으로 작업할 수 있도록 최적화되어 있습니다.

### 10.8 데이터 관리

**Q: 내 데이터를 백업할 수 있나요?**

A: 현재 버전에서는 사용자가 직접 백업하는 기능이 없습니다. 시스템 관리자에게 문의하세요.

**Q: 삭제한 업무는 언제까지 보관되나요?**

A: 삭제된 업무는 영구 삭제되지 않는 한 계속 보관됩니다. "삭제된 작업" 메뉴에서 언제든지 확인하고 복원할 수 있습니다.

### 10.9 문제 해결

**Q: 페이지가 로딩되지 않아요.**

A: 다음을 시도해보세요:
1. 페이지 새로고침 (F5 또는 Ctrl+R)
2. 브라우저 캐시 삭제
3. 다른 브라우저로 시도
4. 관리자에게 문의

**Q: 업무가 저장되지 않아요.**

A: 다음을 확인하세요:
1. 인터넷 연결 상태
2. 브라우저 콘솔 오류 메시지
3. 필수 입력 항목이 비어있지 않은지
4. 권한이 있는지 (조회 권한만 있으면 저장 불가)

**Q: 슬라이드오버 패널이 열리지 않아요.**

A: 다음을 시도해보세요:
1. 페이지 새로고침
2. 다른 업무 클릭 시도
3. 브라우저 팝업 차단 해제 확인
4. 브라우저 확장 프로그램 비활성화 시도
