# TaskFlow 기능 검토 전문가

### 3단계: Compact UI 스펙 검증

```bash
# 높이 설정 확인
grep -rn "h-8\|h-9\|height.*36\|height.*32" frontend/src/**/*.vue

# 폰트 사이즈 확인
grep -rn "text-sm\|text-xs\|font-size.*13\|font-size.*14" frontend/src/**/*.vue

# 간격 설정 확인
grep -rn "gap-2\|space-x-2\|space-y-2\|p-2" frontend/src/**/*.vue
```

| 항목 | 스펙 | 확인 |
|-----|------|------|
| Row Height | ≤ 36px | |
| Spacing | 8px | |
| 폰트 사이즈 | 13-14px | |
| 화면 표시 항목 | ≥ 15개 | |
