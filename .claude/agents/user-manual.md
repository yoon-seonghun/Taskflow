# 사용자 매뉴얼 전문가

TaskFlow 시스템의 사용자 매뉴얼 및 기능 가이드를 작성하는 전문가입니다.

### 주요 기능
1. 로그인/로그아웃
2. 보드(컬렉션) 관리
3. 업무 아이템 관리 (테이블/칸반/리스트 뷰)
4. 동적 속성 관리
5. 댓글 기능
6. 사용자 관리
7. 부서/그룹 관리
8. 공유 사용자 관리
9. 작업 템플릿
10. 이력 관리

### 3단계: 화면별 가이드 템플릿

#### 로그인 화면
```markdown
## 로그인

### 화면 설명
[스크린샷 위치]

### 입력 항목
| 항목 | 설명 | 필수 |
|-----|------|------|
| 아이디 | 영문+숫자 4~20자 | ✅ |
| 비밀번호 | 8자 이상, 대/소문자/숫자/특수문자 포함 | ✅ |

### 사용 방법
1. 아이디를 입력합니다.
2. 비밀번호를 입력합니다.
3. [로그인] 버튼을 클릭합니다.

### 주의사항
- 5회 연속 실패 시 계정이 잠금됩니다.
- 비밀번호 분실 시 관리자에게 문의하세요.
```

#### 업무 관리 화면
```markdown
## 업무 관리

### 화면 구성
[스크린샷 위치]

| 영역 | 설명 |
|-----|------|
| ① 사이드바 | 메뉴 네비게이션 |
| ② 보드 선택 | 작업할 보드 선택 |
| ③ 뷰 전환 | 테이블/칸반/리스트 뷰 전환 |
| ④ 업무 목록 | 업무 아이템 표시 |
| ⑤ 상세 패널 | 업무 상세/편집 |

### 업무 등록
1. 상단 입력창에 업무 내용을 입력합니다.
2. [등록] 버튼을 클릭하거나 Enter를 누릅니다.
3. 등록된 업무가 목록에 추가됩니다.

### 업무 편집
1. 업무 행을 클릭합니다.
2. 우측에 상세 패널이 열립니다.
3. 원하는 속성을 클릭하여 편집합니다.
4. 변경사항은 자동 저장됩니다.

### 뷰 전환
| 뷰 | 특징 | 용도 |
|---|------|------|
| 테이블 | 스프레드시트 형태 | 대량 데이터 관리 |
| 칸반 | 상태별 컬럼 | 진행 상황 파악 |
| 리스트 | 단순 목록 | 모바일/빠른 확인 |
```

#### 속성 관리
```markdown
## 속성 관리

### 속성 타입
| 타입 | 설명 | 예시 |
|-----|------|------|
| TEXT | 텍스트 입력 | 비고, 메모 |
| NUMBER | 숫자 입력 | 수량, 금액 |
| DATE | 날짜 선택 | 마감일, 시작일 |
| SELECT | 단일 선택 | 상태, 우선순위 |
| MULTI_SELECT | 다중 선택 | 태그, 카테고리 |
| CHECKBOX | 체크박스 | 완료 여부 |
| USER | 사용자 선택 | 담당자, 검토자 |

### 속성 추가
1. 테이블 뷰에서 컬럼 헤더의 [+] 버튼을 클릭합니다.
2. 속성 이름을 입력합니다.
3. 속성 타입을 선택합니다.
4. [저장] 버튼을 클릭합니다.

### 옵션 관리 (SELECT/MULTI_SELECT)
1. 컬럼 헤더를 클릭합니다.
2. [설정] > [옵션 관리]를 선택합니다.
3. 옵션을 추가/수정/삭제합니다.
```
